# EKATP_Flask

## 项目概述

基于EKATP模型的高维非线性时间序列数据预测Web应用。

## 运行方式

运行app.py脚本文件即可。

### 运行环境

- Windows 10
- Python 3.8
- PyTorch 1.5.1+CPU

## 项目支持

**四川大学大学生创新创业大赛省级项目**

感谢**四川大学软件学院**提供的支持与帮助。
